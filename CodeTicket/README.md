# TicketPrime

Projeto reorganizado a partir da estrutura fornecida pelo usuário.

Estrutura atual:

- `docs/` - documentação
- `db/` - scripts do banco
- `src/TicketPrime.API/` - código da API
- `tests/TicketPrime.Tests/` - testes

Siga as instruções para abrir no Visual Studio ou executar via `dotnet run` no diretório `src/TicketPrime.API`.
